\section{Contexto.}

El contexto de este proyecto de SSII se engloba dentro del amplio mundo del \textit{Aprendizaje Automatico}
y del \textit{Reconocimiento de formas}.

\begin{itemize}

\item{Sistema A: Lectura e interpretación del contenido de un fichero DWG.}
\item{Sistema B: Visualización y manipulado del contenido de un fichero DWG.}

\end{itemize}

Este proyecto se centra en el contexo del sistema A. El sistema B será desarrollado por otro alumno de la asignatura 
Sistemas Informáticos y ambos sistemas deberán cooperar para cubrir las funcionalidades completas del proyecto.

Este trabajo dará lugar también al proyecto fin de carrera de Ignacio Robledo Rosell. A lo largo del documento se 
describen algunas de las líneas de trabajo futuras hacía las que puede evolucionar el sistema con motivo del desarrollo
del proyecto fin de carrera.
